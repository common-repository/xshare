== Description ==

This plugins easily lets you add social share and Buffer buttons on your website. It has a powerful options panel in WordPress admin area. A user can on or off any option by toggling switch button. xShare also count the number of total shares of a url of your website page on Facebook, Twitter, Pinterest and Linkedin.

To configure the settings: Go to Dashbaord -> xShare

= Available Platforms =
1. **Facebook**
2. **Twitter**
3. **Linkedin**
4. **Pinterest**
5. **Buffer**

= Compatibility =
xShare is compatible with any WordPress theme, whether created by us or by a third party developer.

We are open for any suggestion for this plugin. Feel free to let us know [here](https://rainastudio.com/contact/).

== Installation ==

1. Upload the entire genesis-topbar folder to the /wp-content/plugins/ directory (or install directly by searching in Plugins > Add New)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. In the WordPress Admin area you'll see menu "xShare", add or modify content for your topbar from the meta fields.

= Who should Use xShare plugin? =
xShare plugin is perfect for business owners, marketers, bloggers, influencers, agencies, and basically anyone else who wants to their website page get shared on social media platform.

Nope! We understand that many WordPress users aren't technical and so have built all of our plugins to be as easy as possible to set up and use. The easy feed setup process just requires you to login into your website and then navigate to WordPress Dasboard > xShare to setting up as you want.

The plugin is specifically built with speed and performance in mind. We dynamically load data as needed to ensure that your page speed stays lightning fast.

= Do xShare plugin work on non-Genesis Framework sites? =
Yes, the plugin is primarily designed to work with all themes and frameworks.
